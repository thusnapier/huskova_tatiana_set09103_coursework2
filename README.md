# huskova_tatiana_set09103_coursework2
A guide how to run the web-app in Levinux environment:

Launch Levinux and wait until it loads
Launch PuTTY, open new session and insert into: Host Name: tc@localhost Port: 2222 Connection type: SSH
Type in password: foo
Find the folder in which assignemnt is located (huskova_tatiana_set09103_coursework2)
Launch 'index.py' by typing: 'python index.py'
Wait until everything loads
Go to the browser and type in: http://localhost:5000
